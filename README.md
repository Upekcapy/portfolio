# Upek Perera Portfolio

A polished one-page portfolio for Upek Perera, a Toronto Metropolitan University Computer Science student applying for software developer co-op roles.

## Highlights

- Built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Lucide icons.
- Includes project sections for NotiStock, NotiStock Android, Rock Paper Scissors, and Hangman.
- Uses custom profile and project assets in `public/images`.
- Designed with a green, white, black, and grey palette with dynamic motion and responsive layouts.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Scripts

```bash
npm run dev
npm run lint
npm run build
```

## Project Structure

- `src/app/page.tsx` renders the portfolio page.
- `src/components/PortfolioPage.tsx` contains the portfolio content, cards, links, and animations.
- `public/images` stores the profile photo and project visuals.
